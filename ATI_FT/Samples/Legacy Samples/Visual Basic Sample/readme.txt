The ATIDAQFT Windows demo source code is provided to you as an example of using
the ATIDAQFT ActiveX component in Visual Basic 6.

Before you can use the source code, you must install the following:

-You must install the demo binaries before you can use the source code.  This
ensures that the ActiveX component is available for the project to reference.

-You must install Traditional NI-DAQ from National Instruments.  The latest
version of Traditional NI-DAQ is 7.4, and is available from www.ni.com.
